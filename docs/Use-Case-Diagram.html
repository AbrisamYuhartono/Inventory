Deskripsi Fitur Peminjaman BMN:

Sistem peminjaman BMN memungkinkan user untuk mengajukan permintaan peminjaman barang inventaris. Admin dapat menyetujui atau menolak permintaan, mengelola status peminjaman, dan menghasilkan dokumen peminjaman. Superadmin memiliki akses penuh untuk mengelola seluruh sistem termasuk user, item, dan ruangan.

Deskripsi Fitur Request Perbaikan Barang:

Sistem perbaikan barang memungkinkan user untuk mengajukan permintaan perbaikan untuk barang yang rusak. User dapat mengupload foto kerusakan dan menentukan tingkat urgensi. Admin dapat menyetujui permintaan, mengelola vendor perbaikan, dan melacak biaya. Sistem ini mendukung workflow lengkap dari pengajuan hingga penyelesaian perbaikan.
