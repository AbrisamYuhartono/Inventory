Deskripsi Fitur Peminjaman BMN:

Sistem peminjaman BMN memungkinkan user untuk mengajukan permintaan peminjaman barang inventaris dengan memilih dari daftar barang yang tersedia dan menentukan pegawai yang akan meminjam. Admin dapat menyetujui atau menolak permintaan, mengelola status peminjaman, dan menghasilkan dokumen peminjaman. Superadmin memiliki akses penuh untuk mengelola seluruh sistem termasuk user, pegawai, item, dan ruangan.

Deskripsi Fitur Request Perbaikan Barang:

Sistem perbaikan barang memungkinkan user untuk mengajukan permintaan perbaikan untuk barang yang rusak dengan memilih dari daftar barang dan menentukan pegawai yang mengajukan. User dapat mengupload foto kerusakan dan menentukan tingkat urgensi. Admin dapat menyetujui permintaan, mengelola vendor perbaikan, dan melacak biaya. Sistem ini mendukung workflow lengkap dari pengajuan hingga penyelesaian perbaikan.

Deskripsi Fitur Lainnya (Updated Focus):

Fitur pendukung sistem dengan fokus utama pada Generate QR Code dan Export Reports. Generate QR Code memungkinkan pembuatan kode QR untuk setiap barang inventaris untuk memudahkan tracking dan identifikasi. Export Reports menyediakan berbagai format laporan (PDF, Excel) untuk analisis dan dokumentasi. Fitur lainnya meliputi autentikasi user, dashboard informatif, pencarian item, notifikasi, backup data, pengaturan sistem, manajemen user, audit logs, dan maintenance database.

üì± Generate QR Code

Sistem otomatis menghasilkan QR Code unik untuk setiap barang inventaris. QR Code dapat di-download dalam format PNG dan digunakan untuk tracking, identifikasi cepat, dan integrasi dengan sistem mobile scanning.

üìä Export Reports

Fitur export laporan komprehensif dalam format PDF dan Excel. Mencakup laporan inventaris, peminjaman, perbaikan, utilisasi barang, dan analisis per unit. Dapat diakses langsung dari header untuk kemudahan penggunaan.
