Daftar Use Case Admin

Login ke Sistem

Autentikasi menggunakan username dan password dengan hak akses admin

Kelola Ruangan

Tambah, edit, hapus ruangan dan mengatur PIC (Person in Charge)

Kelola Inventaris

CRUD inventaris, update status, generate QR code, dan tracking lokasi

Lihat Inventaris

Melihat semua inventaris dengan filter dan pencarian

Setujui Peminjaman

Approve/reject permintaan peminjaman dari user dengan alasan

Kelola Peminjaman

Monitoring peminjaman, tracking overdue, dan pengembalian

Generate Laporan

Membuat laporan inventaris, peminjaman, dan aktivitas

Lihat Dashboard

Melihat statistik dan overview sistem inventaris

Export Data

Export data inventaris dan laporan ke format PDF/Excel

Track Overdue

Monitoring dan follow-up item yang terlambat dikembalikan

Kelola QR Code

Generate dan manage QR code untuk inventaris

Kelola Notifikasi

Mengirim notifikasi reminder dan overdue kepada user
